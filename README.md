Ein praxisnahes Automatisierungsprojekt zur Optimierung betriebsinterner Kalkulationsprozesse.

Status: Eigenständiger Entwurf und Pilotprojekt.

Hintergrund: Ablösung einer selbst optimierten komplexen Excel-basierten Kalkulation durch ein Python-Skript zur Steigerung der Effizienz und Datensicherheit.

Kernfokus: Schnelle Zuordnung von prozentualen Kostenanteilen bei unterschiedlichen Fee´s & Anlagen-/Leistungsarten innerhalb einer Eigentümergemeinschaft, sowie einer GUI (Benutzeroberfläche) zur intuitiven Bedienung per Dropdown

Ausblick: Implementierung einer SQL-Datenbankanbindung und Ausweitung auf mehrere Eigentümergemeinschaften

Info: In der angelegten Eigentümergemeinschaft sind die Anlagenarten zu verschiedenen Anteilen prozentual aufgeteilt, daher ist es aktuell etwas umständlich die genaue Aufteilung aus einer langen Excel zu Suchen. Durch die Optimierung der Datenstruktur und den Einsatz von Excel-Formeln konnten die erforderlichen manuellen Eingaben bereits im Vorfeld von 11 auf 2 reduziert werden. Das Python-Skript automatisiert diesen Prozess nun vollständig.
